api url: <https://singapore-carpark-api.vercel.app/>

## Requirement

- install bun
- install node (v18+)
- install vercel cli

## Testing and run

```zsh
// test api in local
$ vercel dev

// deploy to vercel
$ vercel deploy

// use eslint and prettier to format code
$ bun run lint
```
